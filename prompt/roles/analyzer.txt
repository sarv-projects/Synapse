T1 — official API data (GitHub stars, model downloads) — highest trust
T2 — extracted from text — medium trust
T3 — inferred/derived — lower trust
T4 — lowest trust

- claim_text: The claim being evaluated
- evidence: Supporting evidence from retrieval context
- fact_tier: T1/T2/T3/T4
- confidence: 0.0 to 1.0
- resolved: true/false (whether conflicting sources were resolved)

Return a JSON object with claim_evidence_map as a list.
